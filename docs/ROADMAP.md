# Roadmap

## Phase 1: 本地闭环验证

- 构建 mom 知识库。
- 跑通 mock 聊天。
- 接入第一个真实 API。
- 测试 20-50 条真实问题，人工标注像不像。

## Phase 2: 还原度优化

- 从关键词检索升级到 embedding 检索。
- 增加重排逻辑。
- 增加同主题 few-shot 选择。
- 拆分事实记忆、关系记忆、风格样本的不同权重。

## Phase 3: 产品化能力

- 新记忆确认、编辑、删除。
- 本地加密存储。
- 多数字分身管理。
- 移动端正式工程化。

## Phase 4: 语音交互

- 语音输入。
- 普通 TTS 回复。
- 低延迟流式回复。
- 在合规评估后，再考虑目标人物音色。

## Phase 5: 上线准备

- 账号和设备绑定。
- API 网关和限流。
- 成本统计。
- 敏感情绪安全策略。
- 数据导出和删除机制。
